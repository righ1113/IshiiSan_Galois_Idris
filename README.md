### 定義1.3 群の定義
```idris
infixl 6 <+>
interface Group a where
  (<+>)  : a -> a -> a
  gUnit  : a
  gInv   : a -> a
  vAssoc : (l, c, r : a) -> l <+> (c <+> r) = (l <+> c) <+> r
  vUnit  : (r : a) -> (r <+> gUnit = r, gUnit <+> r = r)
  vInv   : (r : a) -> (r <+> gInv r = gUnit, gInv r <+> r = gUnit)
```
- 群の台集合の元が、値(1とか)の場合と、演算(30度回転とか)の場合がある。
